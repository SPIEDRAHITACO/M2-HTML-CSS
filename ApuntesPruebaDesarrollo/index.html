Dividir imagen de texto

Aquí estamos dividiendo el espacio en dos partes: a la izquierda va este texto y a la derecha va una imagen. Esto se logra usando Flexbox en el contenedor padre.

El contenedor de esta sección usa display: flex;, así el texto y la imagen se colocan en columnas, uno al lado del otro.

¿Qué es deploy?

Hacer deploy es tomar tu proyecto (por ejemplo este index.html y style.css) y subirlo a Internet para que cualquier persona pueda verlo entrando a un enlace (URL).

Una forma muy sencilla de hacer deploy es usando GitHub Pages. Los pasos básicos son:

  1. Crea un repositorio en GitHub y sube tus archivos index.html y style.css.
  2. Ve a Settings (Configuración) del repositorio.
  3. Busca la sección "Pages" o "GitHub Pages".
  4. Elige la rama donde está tu código, normalmente main, y la carpeta /root.
  5. Guarda los cambios y espera unos segundos a que GitHub genere una URL pública.

Esa URL será el enlace que puedes compartir, y allí estará tu página ya deployada en Internet.
